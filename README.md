# pypgx
PyPGx is a Python package for pharmacogenomics (PGx) research.

# Running
```sh
$ pypgx -h
usage: pypgx [-h] tool ...

positional arguments:
  tool        name of tool
    pgkb      extract CPIC guidelines using PharmGKB API

optional arguments:
  -h, --help  show this help message and exit
```
